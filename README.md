# Machine-Learning-Projects
This repository contains projects I have created throughout my post-secondary experience in Machine Learning Analyst.
Traffic Volume Analysis & Forecasting:

Built regression models to analyze and predict annual motor vehicle traffic using road characteristics, time-based features, and local authority data. Applied extensive feature engineering, regularization, and model comparison to balance accuracy and generalization. Results were communicated through interactive visualizations to highlight long-term trends and regional differences.

National Pollutant Release Inventory (NPRI) Forecasting:

Analyzed historical pollutant release data across Canadian provinces to identify long-term environmental trends and forecast future emissions. Aggregated facility-level records, handled reporting inconsistencies, and applied regression models to support environmental monitoring and policy-oriented insights.
